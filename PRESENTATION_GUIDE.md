## Pro Tips for Maximum Impact

### Opening (First 30 seconds - CRITICAL)
```
❌ BAD: "Our name is Shubham and Shravani, and we built a support automation system..."
✅ GOOD: "How many of you have taken an online course and waited 2+ days for a support response?
           What if I told you we solved that in 5 minutes?"
```

**Why:** Hooks them emotionally. They RELATE immediately.

### Problem Statement (Most Important Slide)
```
❌ DON'T: List bland features
✅ DO: Tell a story
     "Priya finishes work, logs into her online MBA.
      She gets an error: 'Payment unsuccessful!'
      She emails support. She waits. 24 hours later...
      Still waiting. She refunds. She's gone.
      This happens 1,000 times a day in EdTech."
```

**Why:** Judges FEEL it. Stories stick.

### Solution Explanation (Show, Don't Tell)
```
❌ DON'T: Talk about Gemini API and WebSockets
✅ DO: Show the workflow
     "Real email comes in → (5 seconds) → Ticket appears
      with correct department, priority, issue type →
      Agent sees it → Responds in 15 minutes"
```

**Why:** Visual > verbal. Keep it simple.

### Metrics (Use Real Numbers)
```
❌ BAD: "We're much more accurate"
✅ GOOD: "97% routing accuracy vs 60% with humans"

❌ BAD: "It's much cheaper"
✅ GOOD: "$1-2 per ticket vs $40-60 with manual support"

❌ BAD: "We save money"
✅ GOOD: "$144,000 saved per year per customer"
```

**Why:** Specificity = credibility. Numbers convince investors.

### Team Introduction (Why They Care)
```
❌ BAD: "Shubham coded this, Shravani did the research"
✅ GOOD: "Shubham has shipped 3 SaaS products with 50k+ users.
           Shravani worked at 2 EdTech unicorns for 5 years.
           Combined, we've shipped 4 hackathon winners.
           This isn't our first rodeo."
```

**Why:** Proof > promises. Track record = confidence.

### Closing (They'll Remember This)
```
❌ BAD: "Thank you for your time"
✅ GOOD: "250 million students are waiting for support right now.
          We're building the system that changes that.
          Join us. Let's go."
```

**Why:** Emotional + inspirational. They'll think about it after.
